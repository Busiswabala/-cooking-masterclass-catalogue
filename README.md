# -cooking-masterclass-catalogue
Project Overview

The Cooking Masterclass Catalogue is a Vue.js single-page application that allows users to browse a collection of cooking courses offered by professional chefs. The application displays course information such as the title, chef, skill level, price, and availability status. Users can save available courses to a wishlist, and the total number of saved courses is displayed in the header. The project demonstrates the use of Vue.js components, props, events, reactive data, and responsive design to create an interactive and user-friendly course catalogue.
